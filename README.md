# 🧮 Advanced Console-Based Calculator (Java)

## 📌 Overview

This is a **menu-driven console-based calculator application** built using Java.
It performs both **basic arithmetic operations** and **advanced utilities** like temperature and currency conversion with high precision.

---

## 🚀 Features

### ➕ Basic Operations

* Addition
* Subtraction
* Multiplication
* Division (with zero handling)

### 🔢 Advanced Operations

* Square Root
* Power (Exponentiation)

### 🌡️ Temperature Conversion

* Celsius → Fahrenheit
* Fahrenheit → Celsius

### 💱 Currency Conversion

* INR → USD (using static exchange rate)

### ⚠️ Error Handling

* Handles invalid inputs using `InputMismatchException`
* Prevents division by zero
* Validates negative inputs for square root

### 🎯 Precision

* Uses `BigDecimal` for accurate calculations and rounding

---

## 🛠️ Technologies Used

* Java
* Scanner (User Input)
* BigDecimal (Precision Handling)

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/Yasaswini64/Enhanced-Console-based-Calculator.git
```

2. Navigate to the project folder:

```
cd Enhanced-Console-based-Calculator
```

3. Compile the program:

```
javac Calculator.java
```

4. Run the program:

```
java Calculator
```
